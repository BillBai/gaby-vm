让 iOS 成为 gaby-vm 持续测试的一等平台：Xcode runner、共用测试逻辑、CI 门禁与 benchmark 报告。

schema: spec-driven created: 2026-06-22 7 份源文档 → 6 页可视化

变更要点（跨文档摘要）

  1. 新增 ios-runner/：Xcode Host App + XCTest，Simulator（CI）与真机（本地）跑 correctness 与 benchmark。
  2. Callable entry 重构：host CTest 与 iOS XCTest 共用同一套 suite/bench 逻辑。
  3. CI 加 ios-sim-test 确定性门禁；bench.yml 加 iOS report-only；非 arm64 必须显式 SKIP。
  4. 库通过 CMake 生成的 Xcode subproject 集成；fallback 为 direct-compile + drift guard；不用预编译 .a。
  5. 不改 shipped API、VIXL boundary 或解释器语义；真机 CI 与 App Store 分发不在范围内。
